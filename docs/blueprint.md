# **App Name**: Somobai Pay

## Core Features:

- Homepage: Homepage with organization details and key links.
- Navbar: Navigation bar with organization logo/name and links to My Account, Loan Status, and Payment sections.
- My Account: User profile section displaying personal details and payment history.
- Loan Status: Loan status section, will show loan related info of a specific user.
- Payment Section: Payment gateway integration via bKash for monthly dues (Bangladeshi Tk currency).
- User total deposit: Display the sum total amount deposited by each user.
- Organization fund total: Display the total organization fund amount (total from all users' payments).

## Style Guidelines:

- Primary color: Deep sky blue (#3EB489), a vibrant and trustworthy color evoking the Bay of Bengal.
- Background color: Light grayish-blue (#E0F4F1) to give a clean, professional background.
- Accent color: Saturated yellowish-green (#A6D977) to signal key CTAs like payment buttons.
- Font pairing: 'Poppins' (sans-serif) for headings and 'PT Sans' (sans-serif) for body text to maintain readability and a modern aesthetic.
- Use clear and recognizable icons for navigation and actions, maintaining consistency.
- Clean, tabular layouts to clearly display payment histories and financial data.
- Subtle transitions for page loading and user interactions to enhance user experience.
